request url response
GET request / return welcome to the site
GET request /movies Gets all information about all movies
GET request /movies/:title Gets movie by title
GET request /movies/genre/:name return info about a specific movie genre
GET request /movies/director/:name return info about a specific movie director
GET request /users Get all users
POST request /users register to our system
PUT request /users/:ID Update the persons user info
POST request /users/:ID/:movieID Add a movie to a specific user's profile
DELETE request /users/:id/unregister Unregister a user from the database
DELETE request /users/:ID/:deleteFavorite Delete a favorite movie from the users favorites
